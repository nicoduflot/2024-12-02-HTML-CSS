Les animations

Animate Css

Animate CSS est une bibliothèque de d'animation CSS prêtes à l'emploi. On récupère le CSS, on le lie à la page d'usage, ensuite, on lit la doc pour ajouter aux éléments a animer les classe CSS nécessaires.

Faire ses animations

Pour faire ses animations, on utilise la proriété CSS animation, qui possède des "sous-propriétés". On règle la durée, le style, etc. de l'animation, on indique ensuite le nom du lot d'images clef, c'est à dire l'animation en elle même.
